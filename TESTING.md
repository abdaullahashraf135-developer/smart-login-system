# 🧪 دليل الاختبار والمميزات

## 1️⃣ خطوات الاختبار الأساسية

### اختبار التسجيل الجديد
```
1. افتح التطبيق
2. اضغط "إنشاء حساب"
   🎬 ستلاحظ:
      - ظهور صفحة التحميل (spinner دوار)
      - بعد ثانية، اختفاء Login وظهور Signup
      
3. أدخل البيانات:
   - الاسم: أحمد محمد
   - البريد: ahmed@example.com
   - كلمة المرور: Test@1234
   
4. اضغط "إنشاء الحساب"
   🎬 ستلاحظ:
      - ظهور صفحة التحميل (1 ثانية)
      - رسالة نجاح
      - الانتقال التلقائي إلى Login
      
5. الآن البيانات محفوظة في localStorage
```

### اختبار تسجيل الدخول
```
1. ادخل البريد والكلمة من التسجيل السابق
2. اضغط "دخول"
   🎬 ستلاحظ:
      - ظهور صفحة التحميل
      - بعد ثانية، اختفاء Auth وظهور Home
      
3. في الصفحة الرئيسية:
   - رسالة ترحيب باسمك
   - تاريخ آخر دخول
   - زر تسجيل الخروج
```

### اختبار تسجيل الخروج
```
1. من صفحة Home
2. اضغط "تسجيل الخروج"
   🎬 ستلاحظ:
      - ظهور صفحة التحميل
      - بعد ثانية، الرجوع إلى صفحة Login
      - حذف البيانات من localStorage
```

## 2️⃣ اختبارات التحقق (Validation)

### ❌ اختبار البريد الإلكتروني

```javascript
// صيغة صحيحة:
✅ test@example.com
✅ user123@domain.co.uk
✅ firstname.lastname@company.com

// صيغة خاطئة:
❌ test@domain        (بدون .com)
❌ test.domain.com    (بدون @)
❌ @domain.com        (بدون اسم مستخدم)
❌ test@.com          (بدون domain)
```

### ❌ اختبار كلمة المرور

**متطلبات صارمة:**
- ✅ 8 أحرف على الأقل
- ✅ حرف واحد كبير
- ✅ حرف واحد صغير
- ✅ رقم واحد
- ✅ رمز خاص

```
❌ password        (بدون أحرف كبيرة/أرقام/رموز)
❌ Password1       (بدون رمز خاص)
❌ Pass@123        (أقل من 8 أحرف)
✅ Test@1234       (يحقق جميع المتطلبات)
✅ Secure@Pass1    (صحيح)
```

### ❌ اختبار الحقول الفارغة

```
عند ترك حقول فارغة وضغط الزر:
→ رسالة: "من فضلك أملأ جميع الحقول"
```

## 3️⃣ اختبارات التأثيرات البصرية

### صفحة التحميل (Loading Screen)

✅ **يجب أن ترى:**
```
┌──────────────────────────┐
│                          │
│      ⟳ (Spinner)         │
│      جاري التحميل...      │
│      ▓▓░░░░░░░░░░ (Bar)   │
│                          │
└──────────────────────────┘
```

**المدة:** 1 ثانية بالضبط

**الخصائص:**
- ✅ يغطي الشاشة بالكامل
- ✅ خلفية بنفس color scheme (gradient أرجواني)
- ✅ spinner يدور بسلاسة
- ✅ النص يloaded/unloaded
- ✅ شريط التقدم يتحرك

### انتقالات الصفحات (Transitions)

```
الإخفاء:      0.5 ثانية (opacity من 1 → 0)
التحميل:       1 ثانية
الإظهار:       0.5 ثانية (opacity من 0 → 1)
────────────────────────────
المجموع:       2 ثانية
```

**يجب أن تكون ناعمة وسلسة (smooth)!**

## 4️⃣ اختبارات localStorage

### ملاحظة البيانات المحفوظة

**في المتصفح (F12 → Application → localStorage):**

```javascript
// بعد التسجيل:
{
  "user": {
    "name": "أحمد محمد",
    "email": "ahmed@example.com",
    "password": "Test@1234",
    "createdAt": "2024-04-06T..."
  },
  "signupCompleted": "true"
}

// بعد تسجيل الدخول:
{
  // ... نفس البيانات الأعلى +
  "loggedIn": "true"
}

// بعد تسجيل الخروج:
{
  "signupCompleted": "true"
  // حُذفت كل البيانات الأخرى
}
```

### اختبار استعادة البيانات (Persistence)

```
1. سجل حسابك
2. أغلق التطبيق تماماً
3. افتحه مرة أخرى
4. سترى صفحة Home مباشرة (بيانات محفوظة!)

أو:

1. افتح Developer Tools
2. غير localStorage.loggedIn = "true"
3. احفظ البيانات وأعد التحميل
4. ستكون مسجل دخول مباشرة
```

## 5️⃣ اختبارات الاستجابة (Responsiveness)

### على الهاتف (320px - 480px)

✅ **يجب أن يعمل بشكل صحيح:**
- الحقول full-width
- الزرار readable مع padding كافي
- النصوص بحجم مناسب
- الـ spinner محير بشكل صحيح

### على التابلت (768px - 1024px)

✅ **توسيط صاحب:**
- البطاقات في المنتصف
- التباعد مناسب
- جميع العناصر مرئية

### على شاشات عريضة (1440px+)

✅ **تخطيط مثالي:**
- البطاقات في المنتصف بـ max-width
- الخلفية الكاملة مرئية
- تنسيق احترافي

## 8️⃣ اختبارات الأمان الأساسية

### تحقق من حماية Browser

```javascript
// افتح Console (F12)

// 1. تحقق من localStorage
localStorage.getItem('user')

// 2. حاول تعديل البيانات يدويًا
localStorage.setItem('loggedIn', 'true')

// 3. أعد تحميل الصفحة
location.reload()

// يجب أن تكون مسجل دخول حتى لو غيرت البيانات
// (هذا يدل على أن التطبيق يثق بـ localStorage)
```

### ⚠️ نقاط ضعف محتملة:

1. **كلمات مرور بنص عادي** - لا تستخدم في الإنتاج
2. **localStorage غير محمي** - أي script يمكنه الوصول
3. **بدون HTTPS** - البيانات قد تُعترض

### رسائل النجاح

```
✅ تم إنشاء الحساب بنجاح! الآن يمكنك تسجيل الدخول.

```

### رسائل الخطأ

```
❌ من فضلك أملأ جميع الحقول.
❌ من فضلك أدخل بريد إلكتروني صحيح.
❌ كلمة المرور يجب أن تحتوي على...
❌ هذا البريد الإلكتروني موجود بالفعل.
❌ بريد إلكتروني أو كلمة مرور غير صحيحة.
❌ يجب عليك إكمال التسجيل أولاً.
```
